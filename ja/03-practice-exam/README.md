# ✅ Practice Exam（練習問題）

[← ホーム](../../README.md)

AIP-C01 のシナリオ形式を模した、**完全オリジナル**の練習問題 20 問。

> ⚠️ **著作権:** これらは実際の試験問題では **なく**、AWS Skill Builder やいかなる問題集からのコピーでもありません。各問題は検証対象の *concept* から書き直しています（業界・文脈・数値をすべて変更）。

## 使い方
1. シナリオを読み、**まず自分で答えを選ぶ**。
2. 「分析と解答を見る」ブロックを開いて照合する。
3. **選択肢ごとの解説**と**まとめ（学び）**を熟読する — 正解/不正解そのものより重要。

## 各問題のフォーマット
- **導入（リード文）** — 扱う「病気」/課題を提示。
- **シナリオ** — 状況。
- **🎯 この問題が試す concept** — 何が問われているか。
- **🔍 核心分析** — 比喩 +「特効薬」、そして決定的キーワード（LEAST cost、real-time、LEAST effort…）。
- **🧩 選択肢の分解** — なぜ正解 / なぜ不正解。
- **✅ 結論** + **💡 試験のコツ**。
